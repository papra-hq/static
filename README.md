# Papra static

Host for static generated assets, such as the sponsors SVG.

## Usage

Assets can be accessed using jsDelivr CDN:

- https://cdn.jsdelivr.net/gh/papra-hq/static/assets/sponsors.svg

## Credits

Sponsors are generated using [sponsorkit](https://github.com/antfu-collective/sponsorkit).
